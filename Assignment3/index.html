Welcome to my website designed and built to consolidate my notes made during the lectures of Computing Science module CSC1026 - Web Site Design & Construction. This website covers the main bulk of content covered during part one of this module, dealing with certain Client-Side technologies (HTML, CSS and JavaScript).

These technologies are the building blocks of most websites on the internet, with HTML and CSS both being integral parts of web development and JavaScript being used frequently to make things more interactive.

Also covered on this site is the history of the World Wide Web which, whilst brief, covers all the major points from Tim Berners-Lee's initial concept to the formation of the W3C, and ratification of HTML standards.

HTML (Hyper Text Markup Language) and CSS (Cascading Style Sheets) will be covered in a lot more detail, and we will touch on the basics of JavaScript, including some slightly more intricate functions within external JQuery libraries.
