History of the World Wide Web

The World Wide Web is not the same as the internet. Let's get that out of the way right now. The internet is a series of connected networks which use a particular set of protocols to transfer information from point to point. The World Wide Web (often abbreviated to WWW) is a graphical representation of such information displayed, usually through a bespoke web-browser, in a manner which is easily readable by the user.

Sir Tim Berners-Lee - Father of the World Wide Web

The World Wide Web was fathered by Sir Tim Berners-Lee, once of CERN (yes, that place which threatens to create a black hole every 4 or 5 years), now Director of the World Wide Web Consortium (W3C), the organisation responsible for developing, and formalising, web standards, such as HTML4 and CSS2.

Tim Berners-Lee started working on what would become the World Wide Web in October 1990, as a side project at CERN. By November of the same year, Berners-Lee had created his WorldWideWeb browser and the first web server (still active here). In August 1991 CERN approved the World Wide Web technology and released it to a public FTP server.

By 1993 there were graphical web browsers (including Tim Berners-Lee's own Mosaic browser) available for all major platforms. CERN announced that World Wide Web technology would remain free to use, with no fees in the future. This was a major move leading to the success of the Web, as the lack of a financial barrier to entry meant that anyone, anywhere (as long as they had access to a computer) could develop a web site. Throughout the duration of 1993 the number of HTTP (HyperText Transfer Protocol) servers in use jumped from 26 to over 200.

In October 1994 the World Wide Web Consortium (W3C) was founded at MIT (Massachusetts Institute of Technology, a highly regarded technological education facility) by Tim Berners-Lee, in collaboration with CERN (though Tim Berners-Lee had left CERN by this point).

The world was pretty quiet for the next few years, certain online companies emerged, such as eBay and Amazon (both started in 1995). Microsoft and Netscape argued over whose browser was better. In January 1997 the W3C released their first HTML recommendation, version 3.2. Though there had been other HTML specifications, this was the first that was standardised and recommended openly by the W3C.

HTML 4.0 followed in December and, as a standard, has remained relatively unchanged since (though the HTML 5 release candidate is up for approval right now...), despite a minor update in December 1999 to HTML 4.01.

The Future

Whilst we cannot predict the future with complete certainty, it is fairly likely that the W3C will approve HTML 5 and release the specification as the accepted standard and recommendation. The introduction of HTML5 brings with it a more compact and efficient HTML, interlaced with more eclectic CSS and improved rendering and drawing facilities. Also in the future: Alien attack. Ok, maybe not...

History in Brief

  • 1989 - Tim Berners-Lee proposes HyperText system for Cern.
  • 1990 - WorldWideWeb browser created.
  • 1991 - CERN releases WWW technology to the masses.
  • 1992 - 26 HTTP servers in use
  • 1993 - GUI browsers on all major platforms
  • 1993 - 200+ HTTP servers online.
  • 1994 - W3C founded.
  • 1995 - WWW overtakes FTP as most used internet service.
  • 1998 - Google launches
  • 2002 - Blogging takes off
  • 2005 - YouTube launches
  • 2007 - Google announced as "Most Valuable Brand" and is #1 visited web site.
